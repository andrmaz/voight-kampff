Voight-Kampff — subject & signals

Each step calls a subject model (answers the stimulus in character). There is no second LLM pass. A versioned rubric in this service extracts weighted text features and a composite score — signals for humans to interpret, not proof of sentience. The run manifest records rubric id/version so you can compare runs.

Run step: one subject completion via the API; scoring runs in-process.
